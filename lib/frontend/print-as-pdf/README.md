# @tuleap/print-as-pdf

A library to launch the print command of the browser on a given DOM element, with a PDF template.

## Installation

```
pnpm i @tuleap/print-as-pdf
```

## Usage

### Default usage

```typescript
import { printAsPdf } from "@tuleap/print-as-pdf";

printAsPdf(dom_element, pdf_template);
```
