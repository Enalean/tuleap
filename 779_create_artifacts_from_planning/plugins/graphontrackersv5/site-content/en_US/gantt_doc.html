• Start Date: Date on which the tasks actually starts. May be null .
  • Due Date:Committed date of task achievement. Used as Finish Date if Finish Date is null .
  • Finish Date: Date on which the task is finished or expected to be finished (schedule). May be null (takes Due Date value in that case) .
  • As Of Date: Date considered as a reference for data display. Default value is today.
  • Progress: Percentage of completion of the task.
  • Right text: Text to be displayed at the right of the gantt bar.
  • Hint left, Hint center, Hint right: Text to be displayed on the bar tooltip.
  • Summary text: Text to be displayed on the gantt left, and in the bar tooltip.
  • Scale Can be Year/Month/Week/Day .
