## Various tools that might be usefull for a site admin ##

* switch_ip.sh
  Change IP address in named and httpd configuration

* switch_domain.sh
  Change machine name in all configuration files (see the comments at the end
  of script for details).

* svn_log_parser.php
  This tool parse svn_log files to find the top access repositories
  in the given log file. There is default threshold of 5%.
