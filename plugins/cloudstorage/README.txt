CLOUDSTORAGE PLUGIN
<< TULEAP IN THE CLOUD ! >>


About
-----
This plugin provides the ability to display a view through docman, of your files into a folder hosted on a cloudStorage platforms.


Service available
-----------------
Dropbox and Google Drive.


Prerequisits
------------
The plugin can operate independently, nevertheless, it is strongly recommended to install and/or activate docman plugin to take advantage of the functionality of this plugin.


How to install
--------------
1) (Recommended) Install the latest version of docman including the patch to support this plugin.
2) Install this plugin
3) Make the plugin available in plugins administration page
4) Enable service in project/admin/servicebar...
5) You can now access to cloudstorage plugin via: plugins/cloudstorage/?group_id=1
