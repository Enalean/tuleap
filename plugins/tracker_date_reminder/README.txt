== WARNING ==
This plugin applies ONLY on TRACKERv3

== Description ==

This plugins allows to configure date based reminder on tracker v3.

Once installed, you can configure 'date' fields in trackers to send reminder 
email depending on the values they hold.

Example:
It's useful with a Todo tracker with an "Target completion date" field.
You can set it up to send emails 2 days before artifact's "Target completion date" as a reminder for "Assigned to" people.
