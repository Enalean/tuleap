Good practices:

  • When it is suitable, use the good input type (password, number, email, search, url, â€¦).
  • To disable a form element, use .tlp-form-element-disabled and disabled attribute.
  • To set a form element required, use required attribute and add <i class="fa fa-asterisk"></i> in the label.
  • To put a form element in error, use .tlp-form-element-error.
  • By default, form elements are displayed using width: 100%. You can override this by setting a custom width in css or by adding:
    • the size attribute on inputs when it makes sens (e.g. size="4" for a year)
    • the cols attribute on textareas when it makes sens (you can also play with rows attribute)
    • the .tlp-select-adjusted class on selects and multi-selects
