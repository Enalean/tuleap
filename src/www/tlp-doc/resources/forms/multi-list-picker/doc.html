Multi List pickers are the new Multi Select2. They are meant to be easier to use and to customise while having a similar UX.

Please note: There are predefined templates available by using special data-attributes on option tags:

  • Use data-avatar-url with the value of avatar url to display user avatar. Example: data-avatar-url="https://my-avatar.example.com"
  • Use data-color-value with the value of the color to display a colored circle next to the option label. The color must be either a tlp-color (i.e inca-silver) or in hexadecimal (i.e #8f8f8f). Example: data-color-value="fiesta-red"
You cannot have the both dataset in a same option.

You can also use the items_template_formatter callback to display these templates.



Options:
  • placeholder (default: ""): A string message inviting the user to fill up the field
  • none_value: string (default: null): The none value given on the select. Selected when no values are selected.
  • items_template_formatter (default: none): An asynchronous callback function returning a lit/html template result to customise items templates
    It takes two parameters:
    • html: HTMLTemplateStringProcessor The lit/html tag function to help the developer to build a secured TemplateResult
    • option_value: string The value of the current select box option
    • option_label: string The text content of the current select box option
