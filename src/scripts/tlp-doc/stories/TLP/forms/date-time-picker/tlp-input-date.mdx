# Date/time picker

<code>.tlp-input-date</code>

Date/time picker is based on <a href="https://flatpickr.js.org/">flatpickr</a>.

## Date

<Canvas withToolbar={true} of={DatePickerStories.DatePicker}></Canvas>

#### Typescript example

It requires an HTMLInputElement and a locale string.
```typescript
import { createDatePicker, getLocaleWithDefault } from "@tuleap/tlp-date-picker";

const locale = getLocaleWithDefault(document); // will read the locale string like "en_US" from the document body
const input = document.querySelector("#date-picker");
if (!(input instanceof HTMLInputElement)) {
    return;
}

createDatePicker(input, locale); // returns the date picker object
```

Import the CSS style:
```scss
@use "pkg:@tuleap/tlp-date-picker";
```
