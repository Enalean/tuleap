## Inline Filters

You can filter inline the table by using the <code>filterInlineTable()</code> component.

The <code>input</code> used for the search must have a <code>data-target-table-id</code> attribute.

Cells in which we can search for the term must have the class <code>tlp-table-cell-filterable</code>.

If there aren't any matching entries, then the filter displays the row with the class <code>tlp-table-empty-filter</code>
which should present to the user an empty state.

<strong>Known limitations:</strong> Table sections <code>plugins-list-filterable</code> and
table footer <code>tfoot</code> are not handled yet by the filter.
