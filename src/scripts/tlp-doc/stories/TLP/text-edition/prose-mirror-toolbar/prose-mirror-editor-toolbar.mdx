# Prose Mirror toolbar

<code>createProseMirrorEditorToolbar(document);</code><br/><br/>

## Properties
- <code>controller: ToolbarController</code>: A ToolbarController containing the ToolbarBus to interact with editors.
- <code>text_elements: TextElements | null</code>: Allows to add buttons to format text:
    - It takes four parameters:
        - <code>bold: boolean</code> Add the "bold" button.
        - <code>italic: boolean</code> Add the "italic" button.
        - <code>code: boolean</code> Add the "code" button.
        - <code>quote: boolean</code> Add the "quote" button.
- <code>list_elements: ListElements | null</code>: Allows to add buttons to create lists:
    - It takes two parameters:
        - <code>ordered_list: boolean</code> Add the "ordered_list" button.
        - <code>bullet_list: boolean</code> Add the "bullet_list" button.
- <code>script_elements: ScriptElements | null</code>: Allows to add buttons to format text using special formats:
    - It takes two parameters:
        - <code>subscript: boolean</code> Add the "subscript" button.
        - <code>superscript: boolean</code> Add the "superscript" button.
- <code>link_elements: LinkElements | null</code>: Allows to add buttons to manage images and links:
    - It takes three parameters:
        - <code>link: boolean</code> Add the "link" button.
        - <code>unlink: boolean</code> Add the "remove link" button.
        - <code>image: boolean</code> Add the "image" button.
- <code>style_elements: StyleElements | null</code>: Allows to add a select to style the text:
    - It takes three parameter:
        - <code>text: boolean</code> Add the text option to the select box.
        - <code>headings: boolean</code> Add headings options to the select box. Cannot be used together with the <code>subtitles</code> option.
        - <code>subtitles: boolean</code> Add a subtitle option to the select box. Cannot be used together with the <code>headings</code> option.
        - <code>preformatted: boolean</code> Add the preformatted text option to the select box.
    - When all the options are disabled, then the select won't be added to the toolbar.
- <code>additional_elements: AdditionalElement[] | null</code>: Allows to add custom buttons to the toolbar.
    - An <code>AdditionalElement</code> takes three parameters:
        - <code>target_name: ItemGroupName</code> which targets an existing item group.
            - <code>BASIC_TEXT_ITEMS_GROUP</code> targets the group of items containing the inline text styles buttons (see <code>text_elements</code>).
            - <code>LIST_ITEMS_GROUP</code> targets the group containing the list buttons (see <code>list_elements</code>).
            - <code>LINK_ITEMS_GROUP</code> targets the group containing the links/image buttons (see <code>link_elements</code>).
            - <code>SCRIPTS_ITEMS_GROUP</code> targets the group containing the special scripts format buttons (see <code>script_elements</code>).
            - <code>TEXT_STYLES_ITEMS_GROUP</code> targets the group containing the select box changing the style of the text (see <code>style_elements</code>).
        - <code>position: AdditionalElementPosition</code> defines where to add the element:
            - Use <code>"before"</code> to position your element outside of the target item group, before it.
            - Use <code>"after"</code> to position your element outside of the target item group, after it.
            - Use <code>"at_the_start"</code> to position your element inside the target item group, before its first element.
            - Use <code>"at_the_end"</code> to position your element inside the target item group, after its last element.
        - <code>item_element: HTMLElement</code>: The HTMLElement to add to the toolbar at the desired location.
