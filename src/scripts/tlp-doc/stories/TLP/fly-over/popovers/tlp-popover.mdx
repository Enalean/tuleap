# Popovers

<code>.tlp-popover</code><br/><br/>

<code>createPopover(trigger: HTMLElement, element: HTMLElement, option = \{}): Popover</code>

## Parameters

- <code>HTMLElement</code> trigger
- <code>Element</code> popover content
- (Optional) \{ Object } options.  Contains:
    - \{ String } <code>placement</code>, defaults to <code>"bottom"</code>.
        Overrides corresponding <code>data-placement</code> attribute.
    - \{ String } <code>trigger</code>, defaults to <code>"hover"</code>.
        If <code>"click"</code>, popover is displayed on click instead of mouseover.
        Overrides corresponding <code>data-trigger</code> attribute.
    - \{ Element } <code>anchor</code>, defaults to <code>trigger</code> parameter.
        The arrow of the popover will be attached to the anchor.

<code>popover.hide(): void</code>
Hide the popover programmatically.

<code>popover.destroy():void</code>
Destroys the popover and removes event listeners. The popover will not show up again.

#### Typescript example

```typescript

        document.getElementById("popover-example"),
```
